Typography

Typeface & Color

Arial is the default typeface, with a line-height of 1.42857143 (10:7). If the financial institution has selected a widget heading font in the Admin Platform tool, then Heading 1 will inherit that font.

Font colors are HEX #333333 or #666666. These font colors guarantee compliance with level AA of WCAG 2.0. If used on a white background or on a gray call out, use HEX #EDEDED.

Heading 1 Arial 28px #333333, line-height 40px

Heading 2 Arial 24px #333333, line-height 34px

Subheading Arial 18px #333333, line-height 26px

Main font size Arial 16px #333333 or Arial #666666, line-height 23px

For long paragraph Arial 14px #666666, line-height 20px

Legal text Arial 13px #666666, line-height 19px

Links (#0055CC in case of FI then FI Color) inherit the size of body font size.

Success Heading Arial 24px #2E6F03

Success messages Arial 14px #2E6F03
Error messages 14px #C90202

Accessibility

The default color for links and icons is #0055CC. When tested against a white background (#FFFFFF) and the light gray callout background (#EDEDED), the contrast ratio meets level AA of WCAG 2.0. Financial Institutions may specify the primary accent color for Online Banking. Specifying a color that meets this level will maintain consistent visual contrast for the application. This site offers a way to test and adjust colors: http://snook.ca/technical/colour_contrast/colour.html

Icons

How to use

Icons fall into two categories. They either indicate that the user can take an action, or they draw the attention of the user to an area or message. Icons should only be used for the purposes indicated below. If they are used in other ways, the purpose of the icon may be lost to the user.

If the icon is not universally recognized, then it must have a description associated with the image. When space/hierarchy permit, icons used as buttons should be designed to indicate that the icon can be tapped to take an action.


Color

Accessibility

All images must have an alt attribute (even if the function of the image is purely decorative and alt=“”). When an image contains text that a sighted user can read, convey this with the alt attribute whenever possible. It is best to avoid images that contain text, however, if this cannot be avoided, the alt value must contain the same string of text.
