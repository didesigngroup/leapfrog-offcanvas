Icons fall into two categories:

  1. Those that indicate an action may be taken.
  2. Those that draw attention to something.

Icons should only be used for the meanings indicated on this page. Other meanings may be lost to the user.

Icons that indicate an action should be a very well-known image, i.e. question mark for help, magnifying glass for search. If the icon is not universally known then it must have a description associated with the image.

Icons used as interactive elements should be designed in such a way that their interactive nature is clear to the user and visually distinct from non-interactive iconography.
