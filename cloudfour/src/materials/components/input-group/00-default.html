--- notes: | Bootstrap input groups extend form controls by adding text or buttons before, after, or on both sides of any text-based ``. Use `.input-group` with an `.input-group-addon` or `.input-group-btn` to prepend or append elements to a single `.form-control`. See [Bootstrap input group documentation](http://getbootstrap.com/components/#input-groups) for further details. links: "Bootstrap docs": http://getbootstrap.com/components/#input-groups ---
