--- notes: | The `.modal` class is Bootstrap's modal component. It comes with modifier classes to support _small_ or _large_ widths. #### Classes - `.modal`: The root element containing the overlay and dialog - `.modal-dialog`: The centered dialog container - `.modal-dialog.modal-sm`: Modifier to limit the `.modal-dialog` width - `.modal-dialog.modal-lg`: Modifier to extend the `.modal-dialog` width - `.modal-content`: The dialog content wrapper - `.modal-header`: The top section of dialog content - `.modal-body`: The middle, main section of dialog content - `.modal-footer`: The bottom section of dialog content - `.modal-title`: The heading element for use within `.modal-header` #### Accessibility - Use `role="dialog"` on the `.modal` element. - Use `role="document"` on the `.modal-dialog` element. - Optionally, use `aria-labelledby` and `aria-describedby` on the `.modal` element. links: Bootstrap docs: http://getbootstrap.com/javascript/#modals --- {{!
