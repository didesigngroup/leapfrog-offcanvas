See the [Bootstrap modal demo](http://getbootstrap.com/javascript/#live-demo) for detailed examples of how to use the modal jQuery plugin. #### Plugin hooks - `data-toggle="modal"`: Goes on the triggering element. - `data-dismiss="modal"`: Goes on "close button" elements within the triggered modal. - `data-target="#modal-id"`: (Optional) The `id` of the target modal. This goes on the triggering element if that element is not an anchor. Otherwise, the modal can be triggered with `href="#modal-id"` instead. links: Bootstrap docs: http://getbootstrap.com/javascript/#live-demo Bootstrap plugin: https://github.com/twbs/bootstrap/blob/master/js/modal.js --- {{#extend "modal.default" modalid="modal-demo"}} {{#content "header"}}
