--- notes: | The arrange component, adapted from [SUIT CSS Arrange](https://github.com/suitcss/components-arrange), makes use of CSS table layout to horizontally and vertically arrange a single row of cells. #### Usage Guidelines The `.di-arrange` and `di-arrange__row` elements should only contain `di-arrange__fit` and `di-arrange__fill` child nodes. Unless using the modifier for equal-width cells, only one `di-arrange__fill` should be used per row. The first `di-arrange__fill` will not share the extra space with subsequent `di-arrange__fill` classes. #### Classes + `di-arrange`: Base class for parent element of arranged items + `di-arrange--middle`: Modifier class for middle-aligned cells + `di-arrange--bottom`: Modifier class for bottom-aligned cells + `di-arrange--equal`: Modifier class for equal-width cells + `di-arrange--gutter`: Modifier class for adding a gutter between cells + `di-arrange__fit`: Child class that makes cells shrink-wrap content + `di-arrange__fill`: Child class that makes cells expand to fill remaining space + `di-arrange__row`: Child class for a new row of cells; only necessary for more than one row links: ---
