There are two sizes for toggles: small and large. The use of each depends on the size constraints of the page.

Toggles are used for an option that the user can turn on or off. For example, Yes/No or On/Off.

The selected color for the toggle when turned “ON” is the financial institution’s primary accent color, if this color information is not available then the default color is #666666.

Use the default browser focus state.


Large Switch Active / No State :
ON OFF
Large Switch Off State :
ON OFF
Large Switch Active / No State and Disabled:
ON OFF
Large Switch Off State and Disabled:
ON OFF
Small Switch Active / No State :
ON OFF
Small Switch Off State :
ON OFF
Small Switch Active / No State and Disabled:
ON OFF
Small Switch Off State and Disabled:
ON OFF
 

Usage:

:
                 - sm: optional, if want to have small version
                 - active: optional, if want to have switch On state.
                 - disabled: optional, if disabled the control is required.
                 
                  <div class='di-toggle-switch-container {sm}'>
                  <div class='di-toggle-switch {active} {disabled}'>
                  <span class='label-on '>ON</span>
                  <span class='label-slider' > </span>
                  <span class='label-off '>OFF</span>
                  </div>
                  </div>
