Typography

  • Heading 1 Arial 28px #333333

  • Heading 2 Arial 24px #333333

  •  Success Heading

  • Subheading Arial 18px #333333

  • Main font size Arial #333333 or Arial #666666
  • For long paragraph Arial 14px #666666
  • Legal text Arial 13px #666666
  • Default Links (Hex #0055CC, padding 20px) in case of FI then FI Color
  • Icon Link
  • This is a sentences example with link in it.

  • Heading Link

  • Link as button
  • Success messages
  • Error messages
 

Usage:

For default and full list of options refer to Bootstrap Typography section. 

                    <h1>Heading 1 Arial 28px #333333</h1>
                    <h2>Heading 2 Arial 24px #333333</h2>
                    <h2 class="text-success"><span class="di-icon-checkmark" aria-hidden="true"></span>&nbsp;Success Heading</h2>
                    <h3>Subheading Arial 18px #333333</h3>
                    <span>Main font size Arial 16px #333333</span> <span class="text-light">or Arial 16px #666666</span>
                    <span class="text-long">For sentences Arial 14px #666666</span>
                    <span class="text-legal">Legal text Arial 13px #666666</span>
                    <a href="">Links 14px default to Hex #0055CC in case of Financial Institution primary color then FI Color.</a>
                    <a class="icon" >  <span span class="glyphicon glyphicon-plus" aria-hidden="true"> </span> Icon Link  </a>
